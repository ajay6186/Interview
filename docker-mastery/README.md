# Docker Mastery

30 hands-on exercises across 6 phases — Beginner to Expert.

## How to use
1. Read the `README.md` inside each exercise folder
2. Edit files in `exercise/` to complete the TODOs
3. Check `solution/` when you're stuck
4. Verify with the commands listed in each README

## Prerequisites
- Docker Desktop installed and running
- Basic command line knowledge

## Phases
| Phase | Topic | Exercises |
|-------|-------|-----------|
| 1 | Fundamentals | Dockerfile basics, base images, ENV/ARG, layers, ports |
| 2 | Intermediate | Multi-stage builds, Compose, volumes, networks, health checks |
| 3 | Advanced | Non-root user, compose override, BuildKit secrets, profiles, limits |
| 4 | Patterns | Env files, sidecar, init, blue/green, compose watch |
| 5 | Real World | Production Node.js, PostgreSQL, Nginx proxy, Redis, full stack |
| 6 | Expert | Distroless, multi-arch, private registry, security hardening |
