HTML5 and the Modern Web

Published on March 15, 2024 by the W3C team.

HTML is the standard markup language maintained by the W3C to define the structure of web content and promote open web standards.

The term semantic HTML refers to using HTML elements that convey meaning about the content they contain, not just how it looks.

"Authors are encouraged to use elements, attributes, and attribute values that are appropriate for their content, rather than ones that happen to produce the desired visual appearance."

— W3C HTML 5.2 Specification
