# HTML Mastery — 6-Phase Exercise Project

A structured, hands-on HTML learning path from beginner to expert.
Each phase contains 5 exercises with progressive difficulty.

## How to Use

1. **Open an exercise file** — e.g. `phase-1-fundamentals/01-document-structure/exercise.html`
2. **Read the `<!-- TODO -->` comments** and fill in the missing HTML
3. **Open in a browser** — the page shows green ✓ / red ✗ for each assertion
4. **Check the solution** if stuck — open `solution.html`
5. **Study patterns** — open `examples.html` for 50 worked examples per topic

## Phases

| Phase | Topic | Exercises |
|-------|-------|-----------|
| 1 | Fundamentals | document structure, text content, links/images, lists/tables, basic forms |
| 2 | Semantic HTML | semantic layout, article/section, navigation, figure/media, inline semantics |
| 3 | Forms & Input | input types, form validation, select/datalist, fieldset/legend, advanced forms |
| 4 | Media & Graphics | audio/video, responsive images, SVG, canvas, embedded content |
| 5 | Accessibility | ARIA basics, live regions, keyboard navigation, landmark roles, accessible forms |
| 6 | Advanced | web components, template/slot, dialog/details, data attributes, meta & SEO |

## Assertions

Every exercise uses DOM-query-based checks displayed in the browser:
- **Green ✓** — assertion passed
- **Red ✗ FAIL** — something is wrong with your HTML
