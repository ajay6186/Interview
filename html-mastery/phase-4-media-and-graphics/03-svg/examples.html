Examples 4.3 — Inline SVG

BASIC (1–13)

1. Minimal SVG

<svg xmlns="http://www.w3.org/2000/svg" width="100" height="100">
  <rect width="100" height="100" fill="steelblue">
</svg>

Minimum inline SVG: xmlns, width, height, and a shape.

2. viewBox attribute

<svg viewBox="0 0 100 100" width="200" height="200">
  <circle cx="50" cy="50" r="50" fill="coral">
</svg>

viewBox="minX minY width height" defines the coordinate system. width/height are CSS display size.

3. rect element

<rect x="10" y="10" width="80" height="60"
      fill="steelblue" stroke="navy" stroke-width="2" rx="8">

rx/ry add rounded corners. fill and stroke set color; stroke-width sets border thickness.

4. circle element

<circle cx="50" cy="50" r="40" fill="coral" stroke="red" stroke-width="2">

cx/cy = center, r = radius. Same fill/stroke attributes as rect.

5. ellipse element

<ellipse cx="100" cy="50" rx="90" ry="40" fill="gold">

Like circle but with separate x-radius (rx) and y-radius (ry).

6. line element

<line x1="0" y1="0" x2="200" y2="100" stroke="black" stroke-width="3">

Draws a straight line from (x1,y1) to (x2,y2). stroke is required — fill does nothing for lines.

7. polyline element

<polyline points="10,80 50,20 90,60 130,10 170,70"
          fill="none" stroke="steelblue" stroke-width="3">

A series of connected line segments. fill="none" is common for charts.

8. polygon element

<polygon points="100,10 190,90 10,90"
         fill="gold" stroke="orange" stroke-width="2">

Like polyline but the last point connects back to the first, forming a closed shape.

9. path element — basics

<path d="M 10 10 L 90 10 L 90 90 L 10 90 Z" fill="none" stroke="blue">

M=moveto, L=lineto, Z=closepath. Path is the most powerful SVG element.

10. path curves — Q and C

<path d="M 10 80 Q 100 10 190 80" fill="none" stroke="purple" stroke-width="3">
<path d="M 10 80 C 40 10 160 10 190 80" fill="none" stroke="red" stroke-width="3">

Q = quadratic Bezier (1 control point). C = cubic Bezier (2 control points).

11. text element

<text x="10" y="40" font-size="24" fill="navy" font-family="sans-serif">Hello SVG</text>

x/y position the baseline. Text in SVG is accessible — screen readers can read it.

12. fill and stroke colors

<rect fill="steelblue" stroke="navy" stroke-width="4">
<rect fill="#FF6B6B" stroke="rgba(0,0,0,0.3)" stroke-width="2">
<circle fill="transparent" stroke="currentColor" stroke-width="2">

Accepts any CSS color value. currentColor inherits from CSS color property.

13. SVG as external file

<img src="icon.svg" alt="Settings icon" width="24" height="24">
<!-- vs inline for styling: -->
<svg>...shapes...</svg>

External SVG: simpler HTML, cached, can't be styled with CSS. Inline: fully styleable, no extra request.

INTERMEDIATE (14–26)

14. SVG groups with transform

<g transform="translate(50, 50)">
  <circle cx="0" cy="0" r="30" fill="coral">
  <text x="0" y="5" text-anchor="middle" fill="white">Hi</text>
</g>

g groups elements and applies shared transforms/styles. translate(x,y) moves the origin.

15. rotate and scale transforms

<rect transform="rotate(45 50 50)" x="30" y="30" width="40" height="40" fill="gold">
<rect transform="scale(1.5)" x="10" y="10" width="40" height="40" fill="coral">

rotate(degrees cx cy) rotates around a center point. scale(factor) scales from origin.

16. defs and use

<defs>
  <circle id="dot" r="5" fill="steelblue">
</defs>
<use href="#dot" x="20" y="20">
<use href="#dot" x="50" y="50">
<use href="#dot" x="80" y="20">

defs stores reusable shapes; use instantiates them. Efficient for repeated elements.

17. linearGradient

<defs>
  <linearGradient id="grad" x1="0%" y1="0%" x2="100%" y2="0%">
    <stop offset="0%" stop-color="steelblue">
    <stop offset="100%" stop-color="coral">
  </linearGradient>
</defs>
<rect width="200" height="100" fill="url(#grad)">

linearGradient with stop elements. Reference with fill="url(#id)".

18. radialGradient

<defs>
  <radialGradient id="rgrad" cx="50%" cy="50%" r="50%">
    <stop offset="0%" stop-color="white">
    <stop offset="100%" stop-color="steelblue">
  </radialGradient>
</defs>
<circle cx="100" cy="100" r="100" fill="url(#rgrad)">

Radial gradient emanating from center. Creates sphere-like effects.

19. clipPath

<defs>
  <clipPath id="circle-clip">
    <circle cx="50" cy="50" r="50">
  </clipPath>
</defs>
<image href="photo.jpg" width="100" height="100" clip-path="url(#circle-clip)">

clipPath masks an element to a shape — circular avatar, for example.

20. filter — blur

<defs>
  <filter id="blur">
    <feGaussianBlur in="SourceGraphic" stdDeviation="5">
  </filter>
</defs>
<rect width="100" height="100" fill="steelblue" filter="url(#blur)">

SVG filters apply effects: blur, drop-shadow, color matrix, displacement, etc.

21. filter — drop shadow

<defs>
  <filter id="shadow" x="-20%" y="-20%" width="140%" height="140%">
    <feDropShadow dx="4" dy="4" stdDeviation="3" flood-color="rgba(0,0,0,0.4)">
  </filter>
</defs>
<rect width="80" height="60" fill="white" filter="url(#shadow)">

feDropShadow creates a realistic drop shadow effect on any SVG element.

22. SVG CSS styling

<style>
.icon { fill: currentColor; transition: fill 0.2s; }
.icon:hover { fill: coral; }
</style>
<svg class="icon" viewBox="0 0 24 24" width="24" height="24">
  <path d="M12 2L2 7l10 5 10-5-10-5zM2 17l10 5 10-5M2 12l10 5 10-5">
</svg>

Inline SVG can be styled with CSS. currentColor inherits the parent element's color.

23. SVG CSS animations

<style>
@keyframes spin { to { transform: rotate(360deg); } }
.spinner { transform-origin: 50% 50%; animation: spin 1s linear infinite; }
</style>
<circle class="spinner" cx="50" cy="50" r="40" fill="none"
        stroke="steelblue" stroke-width="4" stroke-dasharray="60 40">

CSS animations work on SVG elements. stroke-dasharray creates a partial arc for spinners.

24. stroke-dasharray and stroke-dashoffset

<circle cx="60" cy="60" r="50"
        fill="none" stroke="steelblue" stroke-width="8"
        stroke-dasharray="314"
        stroke-dashoffset="78">

dasharray sets dash pattern; dashoffset shifts it. Circumference = 2πr ≈ 314 for r=50. Used for progress rings.

25. textPath — text on a curve

<defs>
  <path id="curve" d="M 10,80 Q 100,10 190,80">
</defs>
<text font-size="16" fill="navy">
  <textPath href="#curve">Text flowing on a curved path</textPath>
</text>

textPath renders text along any SVG path. Useful for arc labels in charts.

26. SVG accessibility with title and desc

<svg role="img" aria-labelledby="svg-title svg-desc" viewBox="0 0 100 100">
  <title id="svg-title">Revenue chart</title>
  <desc id="svg-desc">Bar chart showing $1.2M revenue in Q4 2025</desc>
  <!-- shapes -->
</svg>

title and desc make SVG accessible. aria-labelledby references them for screen readers.

NESTED (27–38)

27. Bar chart

<svg viewBox="0 0 300 200" xmlns="http://www.w3.org/2000/svg">
  <!-- Bars -->
  <rect x="20" y="80" width="40" height="100" fill="steelblue">
  <rect x="80" y="40" width="40" height="140" fill="steelblue">
  <rect x="140" y="60" width="40" height="120" fill="steelblue">
  <rect x="200" y="20" width="40" height="160" fill="coral">
  <!-- Labels -->
  <text x="40" y="195" text-anchor="middle" font-size="12">Q1</text>
  <text x="100" y="195" text-anchor="middle" font-size="12">Q2</text>
  <text x="160" y="195" text-anchor="middle" font-size="12">Q3</text>
  <text x="220" y="195" text-anchor="middle" font-size="12">Q4</text>
</svg>

Simple inline bar chart. SVG is ideal for data visualization — scales to any resolution.

28. Donut chart

<svg viewBox="0 0 100 100">
  <circle cx="50" cy="50" r="40" fill="none"
    stroke="#eee" stroke-width="20">
  <circle cx="50" cy="50" r="40" fill="none"
    stroke="steelblue" stroke-width="20"
    stroke-dasharray="75 125"
    transform="rotate(-90 50 50)">
  <text x="50" y="55" text-anchor="middle" font-size="18" fill="navy">75%</text>
</svg>

Donut chart using stroke-dasharray on a circle. stroke-dasharray="percent total-minus-percent".

29. Icon sprite

<!-- In body, hidden -->
<svg style="display:none">
  <symbol id="icon-home" viewBox="0 0 24 24">
    <path d="M10 20v-6h4v6h5v-8h3L12 3 2 12h3v8z">
  </symbol>
  <symbol id="icon-user" viewBox="0 0 24 24">
    <path d="M12 12c2.7 0 4.8-2.1 4.8-4.8S14.7 2.4 12 2.4 7.2 4.5 7.2 7.2 9.3 12 12 12z">
  </symbol>
</svg>
<!-- Usage anywhere -->
<svg width="24" height="24"><use href="#icon-home"></svg>

SVG sprite: define symbols once, reference with <use>. Reduces HTTP requests.

30. Animated progress ring

<svg viewBox="0 0 100 100" width="100" height="100">
  <circle cx="50" cy="50" r="45" fill="none" stroke="#eee" stroke-width="8">
  <circle id="ring" cx="50" cy="50" r="45" fill="none"
    stroke="steelblue" stroke-width="8"
    stroke-dasharray="282" stroke-dashoffset="282"
    stroke-linecap="round"
    transform="rotate(-90 50 50)">
</svg>
<script>
// Animate to 70%: offset = 282 - (282 * 0.7) = 84.6
document.getElementById('ring').style.strokeDashoffset = 84.6;
</script>

Animated progress ring: 2πr = 283. Offset of 0 = full ring; offset = circumference = empty.

31. Morphing path with CSS animation

<style>
@keyframes morph {
  0%   { d: path("M10,50 Q50,10 90,50 Q50,90 10,50"); }
  50%  { d: path("M10,50 Q50,30 90,50 Q50,70 10,50"); }
  100% { d: path("M10,50 Q50,10 90,50 Q50,90 10,50"); }
}
.blob { animation: morph 3s ease-in-out infinite; }
</style>
<path class="blob" fill="coral">

CSS d property animation for morphing SVG paths (Chrome/Edge). Paths must have same number of commands.

32. SVG pattern fill

<defs>
  <pattern id="dots" width="10" height="10" patternUnits="userSpaceOnUse">
    <circle cx="5" cy="5" r="2" fill="steelblue">
  </pattern>
</defs>
<rect width="200" height="100" fill="url(#dots)">

pattern tiles a shape across an area. patternUnits controls the coordinate system.

33. foreignObject for HTML inside SVG

<svg viewBox="0 0 300 100" xmlns="http://www.w3.org/2000/svg">
  <rect width="300" height="100" fill="#f0f0f0" rx="8">
  <foreignObject x="10" y="10" width="280" height="80">
    <div xmlns="http://www.w3.org/1999/xhtml">
      <p>This is <strong>HTML</strong> inside SVG!</p>
    </div>
  </foreignObject>
</svg>

foreignObject embeds arbitrary HTML (xHTML) inside SVG. Useful for wrapping text.

34. Line chart

<svg viewBox="0 0 300 200" xmlns="http://www.w3.org/2000/svg">
  <polyline
    points="0,180 60,120 120,140 180,60 240,80 300,20"
    fill="none" stroke="steelblue" stroke-width="2">
  <polyline
    points="0,180 60,120 120,140 180,60 240,80 300,20"
    fill="rgba(70,130,180,0.1)" stroke="none">

  <polygon
    points="0,180 60,120 120,140 180,60 240,80 300,20 300,200 0,200"

Area chart: polyline for the line, polygon for the filled area underneath.

35. SVG with SMIL animation (deprecated but works)

<circle cx="50" cy="50" r="40" fill="coral">
  <animate attributeName="r" from="40" to="10" dur="1s"
           begin="click" fill="freeze">
  <animate attributeName="fill" from="coral" to="steelblue" dur="1s"
           begin="click" fill="freeze">
</circle>

SMIL animate elements trigger on events. fill="freeze" holds the end state. Prefer CSS/JS animations.

36. Marker arrows on paths

<defs>
  <marker id="arrow" markerWidth="10" markerHeight="7"
    refX="9" refY="3.5" orient="auto">
    <polygon points="0 0, 10 3.5, 0 7" fill="black">

<line x1="10" y1="50" x2="180" y2="50"

ADVANCED (39–50)

39. Programmatic SVG creation

const svg = document.createElementNS('http://www.w3.org/2000/svg', 'svg');
svg.setAttribute('viewBox', '0 0 100 100');
const circle = document.createElementNS('http://www.w3.org/2000/svg', 'circle');
circle.setAttribute('cx', '50');
circle.setAttribute('cy', '50');
circle.setAttribute('r', '40');
circle.setAttribute('fill', 'coral');
svg.appendChild(circle);
document.body.appendChild(svg);

SVG elements must be created with createElementNS using the SVG namespace URI.

40. getBBox for element bounds

const text = document.querySelector('text');
const box = text.getBBox();
// box: { x, y, width, height }
// Draw a bounding rectangle around it:
const rect = document.createElementNS(svgNS, 'rect');
rect.setAttribute('x', box.x - 4);
rect.setAttribute('y', box.y - 4);
rect.setAttribute('width', box.width + 8);
rect.setAttribute('height', box.height + 8);

getBBox() returns the bounding box of any SVG element in the SVG coordinate system.

41. getPointAtLength for path animation

const path = document.querySelector('path');
const length = path.getTotalLength();
let t = 0;
function animate() {
  const pt = path.getPointAtLength(t % length);
  dot.setAttribute('cx', pt.x);
  dot.setAttribute('cy', pt.y);
  t += 2;
  requestAnimationFrame(animate);
}
animate();

Animate an element along an arbitrary SVG path using getPointAtLength.

42. SVG to PNG with canvas

const svg = document.querySelector('svg');
const xml = new XMLSerializer().serializeToString(svg);
const img = new Image();
img.onload = () => {
  canvas.getContext('2d').drawImage(img, 0, 0);
  link.href = canvas.toDataURL('image/png');
};
img.src = 'data:image/svg+xml;base64,' + btoa(xml);

Convert inline SVG to PNG by serializing to XML, drawing on canvas, then exporting.

43. feColorMatrix filter

<defs>
  <filter id="grayscale">
    <feColorMatrix type="saturate" values="0">
  </filter>
  <filter id="sepia">
    <feColorMatrix type="matrix"
      values="0.393 0.769 0.189 0 0
              0.349 0.686 0.168 0 0
              0.272 0.534 0.131 0 0
              0 0 0 1 0">
  </filter>
</defs>

feColorMatrix applies grayscale, sepia, and custom color transformations.

44. feTurbulence for texture

<defs>
  <filter id="noise">
    <feTurbulence type="fractalNoise" baseFrequency="0.9"
                  numOctaves="4" result="noise">
    <feColorMatrix type="saturate" values="0">
  </filter>
</defs>
<rect width="200" height="200" filter="url(#noise)">

feTurbulence generates procedural noise textures — useful for paper, cloud, or marble effects.

45. feDisplacementMap warping

<defs>
  <filter id="wave">
    <feTurbulence type="turbulence" baseFrequency="0.05" result="noise">
    <feDisplacementMap in="SourceGraphic" in2="noise" scale="30"
                       xChannelSelector="R" yChannelSelector="G">
  </filter>
</defs>
<text filter="url(#wave)" font-size="40">Wavy Text</text>

feDisplacementMap uses a noise map to warp pixels — creates wave, ripple, and heat-haze effects.

46. CSS variable theming of SVG

<style>
:root { --brand: steelblue; }
.icon { fill: var(--brand); }
</style>
<svg class="icon" viewBox="0 0 24 24">
  <path d="M12 2L2 7l10 5 10-5-10-5z">
</svg>
<script>
document.documentElement.style.setProperty('--brand', 'coral');
</script>

CSS custom properties work inside inline SVG — change themes with one variable.

47. SVG with ARIA roles for charts

<svg role="img" aria-label="Monthly sales chart"
     viewBox="0 0 400 300">
  <title>Monthly Sales 2025</title>
  <desc>Bar chart showing sales from Jan to Dec 2025</desc>
  <g role="list" aria-label="Monthly bars">
    <rect role="listitem" aria-label="January: $12,000"
          x="10" y="100" width="20" height="100">
  </g>
</svg>

Complex charts need full ARIA markup for screen readers. Consider a table as an alternative.

48. Viewbox pan and zoom with JS

let viewBox = { x:0, y:0, w:500, h:500 };
svg.addEventListener('wheel', e => {
  const scale = e.deltaY > 0 ? 1.1 : 0.9;
  viewBox.w *= scale;
  viewBox.h *= scale;
  svg.setAttribute('viewBox', `${viewBox.x} ${viewBox.y} ${viewBox.w} ${viewBox.h}`);
  e.preventDefault();
});

Implement zoom by scaling the viewBox dimensions; pan by translating viewBox x/y.

49. CSS counter-based SVG text

<style>
.bar::before { content: counter(pct) "%"; counter-increment: pct; }
</style>
<!-- Better: use D3.js or manual text positioning for dynamic charts -->
<svg viewBox="0 0 300 200">
  <text x="50%" y="50%" text-anchor="middle" dominant-baseline="middle"
        font-size="48" fill="steelblue">42%</text>
</svg>

dominant-baseline="middle" vertically centers text. text-anchor="middle" centers horizontally.

</svg>

<!-- Usage -->
<button>
  <svg width="16" height="16" aria-hidden="true"><use href="#i-check"></svg>

Production icon system: sprite hidden with aria-hidden, icons aria-hidden inside buttons with visible text labels.
