Examples 4.4 — Canvas

BASIC (1–13)

1. Minimal canvas element

<canvas id="c" width="300" height="200">Canvas not supported.</canvas>
<script>
const ctx = document.getElementById('c').getContext('2d');
</script>

Always set width/height attributes (not CSS). Fallback text displays in browsers without canvas support.

2. fillRect — filled rectangle

ctx.fillStyle = 'steelblue';
ctx.fillRect(10, 10, 100, 60); // x, y, width, height

fillRect draws a filled rectangle. fillStyle accepts any CSS color string.

3. strokeRect — outline rectangle

ctx.strokeStyle = 'coral';
ctx.lineWidth = 4;
ctx.strokeRect(10, 10, 100, 60);

strokeRect draws only the border. lineWidth controls thickness.

4. clearRect — erase area

ctx.fillStyle = 'steelblue';
ctx.fillRect(0, 0, 300, 200);
ctx.clearRect(50, 50, 100, 60); // punches a hole

clearRect makes pixels transparent. Used for erasing and animation frame clearing.

5. Drawing a circle (arc)

ctx.beginPath();
ctx.arc(100, 100, 60, 0, Math.PI * 2); // cx, cy, r, startAngle, endAngle
ctx.fillStyle = 'coral';
ctx.fill();

arc() draws circular arcs. Full circle: 0 to Math.PI*2. beginPath() starts a new path.

6. Stroke a circle

ctx.beginPath();
ctx.arc(100, 100, 60, 0, Math.PI * 2);
ctx.strokeStyle = 'navy';
ctx.lineWidth = 4;
ctx.stroke();

Use stroke() instead of fill() to draw only the circle outline.

7. Drawing text

ctx.font = 'bold 24px sans-serif';
ctx.fillStyle = 'navy';
ctx.fillText('Hello Canvas!', 20, 50);

ctx.strokeStyle = 'coral';
ctx.strokeText('Outlined', 20, 100);

fillText and strokeText draw text. font follows CSS font shorthand syntax.

8. Text alignment

ctx.textAlign = 'center';       // left | center | right | start | end
ctx.textBaseline = 'middle';    // top | middle | bottom | alphabetic
ctx.fillText('Centered', canvas.width/2, canvas.height/2);

textAlign controls horizontal position relative to x; textBaseline controls vertical position.

9. Drawing a line

ctx.beginPath();
ctx.moveTo(10, 10);
ctx.lineTo(200, 150);
ctx.strokeStyle = 'black';
ctx.lineWidth = 2;
ctx.stroke();

moveTo sets the pen position without drawing; lineTo draws from current position. Always call stroke() to render.

10. Drawing a polygon (triangle)

ctx.beginPath();
ctx.moveTo(100, 10);
ctx.lineTo(190, 190);
ctx.lineTo(10, 190);
ctx.closePath();   // connects back to start
ctx.fillStyle = 'gold';
ctx.fill();
ctx.strokeStyle = 'orange';
ctx.stroke();

closePath() connects the last point to the first, closing the shape.

11. save() and restore()

ctx.fillStyle = 'blue';
ctx.save();          // save state
ctx.fillStyle = 'red';
ctx.fillRect(10, 10, 50, 50); // red
ctx.restore();       // restore state
ctx.fillRect(70, 10, 50, 50); // blue again

save/restore push/pop the full canvas state (fillStyle, transform, clip, etc.) onto a stack.

12. globalAlpha for transparency

ctx.fillStyle = 'steelblue';
ctx.fillRect(10, 10, 100, 100);

ctx.globalAlpha = 0.5;
ctx.fillStyle = 'coral';
ctx.fillRect(60, 60, 100, 100); // 50% transparent
ctx.globalAlpha = 1.0; // reset

globalAlpha affects all subsequent drawing operations. Reset to 1.0 when done.

13. Canvas fallback content

<canvas width="400" height="300">
  <p>Your browser doesn't support canvas.
     <a href="chart-data.html">View the data table instead.</a></p>
</canvas>

Canvas fallback is shown in very old browsers. Provide a text or table alternative for accessibility.

INTERMEDIATE (14–26)

14. Linear gradient

const grad = ctx.createLinearGradient(0, 0, 300, 0); // x0,y0,x1,y1
grad.addColorStop(0, 'steelblue');
grad.addColorStop(1, 'coral');
ctx.fillStyle = grad;
ctx.fillRect(0, 0, 300, 100);

createLinearGradient takes start and end points. addColorStop(position 0-1, color).

15. Radial gradient

const grad = ctx.createRadialGradient(150, 100, 0, 150, 100, 100);
grad.addColorStop(0, 'white');
grad.addColorStop(1, 'steelblue');
ctx.fillStyle = grad;
ctx.fillRect(0, 0, 300, 200);

createRadialGradient(x0,y0,r0, x1,y1,r1) — inner circle to outer circle.

16. Pattern fill

const img = new Image();
img.onload = () => {
  const pattern = ctx.createPattern(img, 'repeat'); // repeat|repeat-x|repeat-y|no-repeat
  ctx.fillStyle = pattern;
  ctx.fillRect(0, 0, 400, 300);
};
img.src = 'tile.png';

createPattern tiles an image, canvas, or video as a fill.

17. Shadows

ctx.shadowColor = 'rgba(0,0,0,0.5)';
ctx.shadowBlur = 15;
ctx.shadowOffsetX = 5;
ctx.shadowOffsetY = 5;
ctx.fillStyle = 'steelblue';
ctx.fillRect(50, 50, 200, 100);

Shadow properties apply to all subsequent drawing. Reset to transparent/0 to disable.

18. translate transform

ctx.save();
ctx.translate(150, 100); // move origin to center
ctx.fillStyle = 'coral';
ctx.fillRect(-50, -50, 100, 100); // draw centered rect
ctx.restore();

translate moves the coordinate origin. Drawing at (−w/2, −h/2) then centers shapes around origin.

19. rotate transform

ctx.save();
ctx.translate(150, 100);
ctx.rotate(Math.PI / 4); // 45 degrees
ctx.fillStyle = 'gold';
ctx.fillRect(-50, -50, 100, 100);
ctx.restore();

rotate() rotates around the current origin. Always translate to the center of rotation first.

20. scale transform

ctx.save();
ctx.scale(2, 0.5); // 2x wide, half height
ctx.fillStyle = 'steelblue';
ctx.fillRect(50, 50, 100, 100);
ctx.restore();

scale(x, y) stretches the coordinate space. scale(-1, 1) mirrors horizontally.

21. Drawing an image

const img = new Image();
img.onload = () => {
  ctx.drawImage(img, 0, 0);             // natural size
  ctx.drawImage(img, 0, 0, 200, 150);   // scaled
  ctx.drawImage(img, 10,10,80,60, 0,0,200,150); // crop+scale
};
img.src = 'photo.jpg';

drawImage has 3 overloads: position, position+size, crop+position+size.

22. Pixel manipulation with getImageData

const imageData = ctx.getImageData(0, 0, canvas.width, canvas.height);
const data = imageData.data; // Uint8ClampedArray [r,g,b,a, r,g,b,a, ...]
for (let i = 0; i < data.length; i += 4) {
  const avg = (data[i] + data[i+1] + data[i+2]) / 3;
  data[i] = data[i+1] = data[i+2] = avg; // grayscale
}
ctx.putImageData(imageData, 0, 0);

Access and modify individual pixels. Each pixel = 4 bytes (RGBA). putImageData writes back.

23. Canvas to image data URL

const dataURL = canvas.toDataURL('image/png');
const dataURLjpg = canvas.toDataURL('image/jpeg', 0.8); // 80% quality
const link = document.createElement('a');
link.href = dataURL;
link.download = 'drawing.png';
link.click();

toDataURL exports the canvas as a base64 data URL. Trigger download by clicking a hidden link.

24. canvas.toBlob for efficient export

canvas.toBlob(blob => {
  const url = URL.createObjectURL(blob);
  const img = new Image();
  img.src = url;
}, 'image/webp', 0.9);

toBlob is more memory-efficient than toDataURL for large canvases.

25. requestAnimationFrame loop

let x = 0;
function draw() {
  ctx.clearRect(0, 0, canvas.width, canvas.height);
  ctx.fillStyle = 'coral';
  ctx.fillRect(x, 50, 60, 60);
  x = (x + 2) % canvas.width;
  requestAnimationFrame(draw);
}
draw();

requestAnimationFrame syncs to display refresh rate (60fps). Always clear before redrawing.

26. Bezier curve path

ctx.beginPath();
ctx.moveTo(10, 100);
ctx.bezierCurveTo(50, 10, 150, 10, 190, 100); // control1, control2, end
ctx.strokeStyle = 'steelblue';
ctx.lineWidth = 3;
ctx.stroke();

bezierCurveTo draws a cubic Bezier curve with two control points.

NESTED (27–38)

27. Rounded rectangle (roundRect)

ctx.beginPath();
ctx.roundRect(20, 20, 160, 100, 16); // x, y, w, h, radius
ctx.fillStyle = 'steelblue';
ctx.fill();

roundRect() is a modern API. For older browsers, use arcTo() to manually build rounded corners.

28. Clock face

const now = new Date();
const hour = now.getHours() % 12;
const min  = now.getMinutes();
// Hour hand
const hAngle = (hour / 12) * Math.PI * 2 - Math.PI / 2;
ctx.beginPath();
ctx.moveTo(cx, cy);
ctx.lineTo(cx + Math.cos(hAngle) * 50, cy + Math.sin(hAngle) * 50);
ctx.lineWidth = 6;
ctx.stroke();

Convert clock hands to radians: (fraction * 2π − π/2) to start from 12 o'clock.

29. Particle system

const particles = Array.from({length: 100}, () => ({
  x: Math.random() * 400, y: Math.random() * 300,
  vx: (Math.random() - 0.5) * 2, vy: (Math.random() - 0.5) * 2,
  r: Math.random() * 4 + 1
}));
function update() {
  ctx.clearRect(0, 0, 400, 300);
  particles.forEach(p => {
    p.x += p.vx; p.y += p.vy;
    if (p.x < 0 || p.x > 400) p.vx *= -1;
    if (p.y < 0 || p.y > 300) p.vy *= -1;
    ctx.beginPath();
    ctx.arc(p.x, p.y, p.r, 0, Math.PI * 2);
    ctx.fill();
  });
  requestAnimationFrame(update);
}

Simple particle system: update positions, bounce off walls, redraw each frame.

30. Hit testing with isPointInPath

const circlePath = new Path2D();
circlePath.arc(150, 100, 60, 0, Math.PI * 2);
ctx.fill(circlePath);

canvas.addEventListener('click', e => {
  const rect = canvas.getBoundingClientRect();
  const x = e.clientX - rect.left;
  const y = e.clientY - rect.top;
  if (ctx.isPointInPath(circlePath, x, y)) {
    console.log('Circle clicked!');
  }
});

Path2D stores reusable paths. isPointInPath tests if a point is inside the path.

31. OffscreenCanvas for web workers

// main.js
const offscreen = canvas.transferControlToOffscreen();
const worker = new Worker('draw-worker.js');
worker.postMessage({ canvas: offscreen }, [offscreen]);

// draw-worker.js
self.onmessage = ({ data }) => {
  const ctx = data.canvas.getContext('2d');
  // draw without blocking main thread
};

OffscreenCanvas moves rendering to a Web Worker, keeping the main thread free for UI.

32. Composite operations

ctx.fillStyle = 'steelblue';
ctx.fillRect(0, 0, 100, 100);

ctx.globalCompositeOperation = 'xor'; // punch hole
ctx.fillStyle = 'coral';
ctx.beginPath();
ctx.arc(70, 70, 50, 0, Math.PI * 2);
ctx.fill();

globalCompositeOperation controls how new drawing interacts with existing pixels: source-over, multiply, screen, xor, etc.

33. Line cap and join styles

ctx.lineWidth = 20;
ctx.lineCap = 'round';    // butt | round | square
ctx.lineJoin = 'round';   // miter | round | bevel

ctx.beginPath();
ctx.moveTo(20, 50);
ctx.lineTo(100, 20);
ctx.lineTo(180, 50);
ctx.stroke();

lineCap styles the ends of open paths; lineJoin styles corners where lines meet.

34. Dashed lines

ctx.setLineDash([10, 5]);      // [dash length, gap length]
ctx.lineDashOffset = 0;        // phase offset for animation
ctx.beginPath();
ctx.moveTo(10, 50);
ctx.lineTo(290, 50);
ctx.stroke();
ctx.setLineDash([]);           // reset to solid

setLineDash accepts an array of dash/gap lengths. Animate lineDashOffset for marching ants effect.

35. Clip region

ctx.save();
ctx.beginPath();
ctx.arc(150, 100, 80, 0, Math.PI * 2);
ctx.clip(); // everything now clipped to circle
ctx.drawImage(img, 0, 0, 300, 200);
ctx.restore();

clip() restricts drawing to the current path shape. restore() removes the clip region.

36. measureText for layout

ctx.font = '24px sans-serif';
const metrics = ctx.measureText('Hello World');
const textWidth = metrics.width;
// Center horizontally:
ctx.fillText('Hello World', (canvas.width - textWidth) / 2, 100);

measureText returns text dimensions for precise layout calculations before drawing.

37. High-DPI canvas

const dpr = window.devicePixelRatio || 1;
canvas.width  = 400 * dpr;
canvas.height = 300 * dpr;
canvas.style.width  = '400px';
canvas.style.height = '300px';
ctx.scale(dpr, dpr); // scale all drawing to match device pixels

Scale canvas buffer by devicePixelRatio for crisp rendering on Retina/HiDPI screens.

38. Canvas as WebGL surface

const gl = canvas.getContext('webgl');
// or 'webgl2' for WebGL 2.0
if (!gl) {
  console.error('WebGL not supported');
} else {
  gl.clearColor(0.2, 0.4, 0.8, 1.0);
  gl.clear(gl.COLOR_BUFFER_BIT);
}

Canvas also supports WebGL for GPU-accelerated 3D graphics via getContext('webgl').

ADVANCED (39–50)

39. Canvas sprite animation

const spriteSheet = new Image();
spriteSheet.src = 'sprite.png';
let frame = 0;
const FRAME_W = 64, FRAME_H = 64, TOTAL_FRAMES = 8;

function animate() {
  ctx.clearRect(0, 0, canvas.width, canvas.height);
  ctx.drawImage(spriteSheet, frame * FRAME_W, 0, FRAME_W, FRAME_H, 0, 0, 64, 64);
  frame = (frame + 1) % TOTAL_FRAMES;
  requestAnimationFrame(animate);
}
spriteSheet.onload = animate;

Sprite animation: advance through frames of a sprite sheet each animation frame.

40. Canvas game loop with delta time

let lastTime = 0;
function gameLoop(timestamp) {
  const delta = timestamp - lastTime; // ms since last frame
  lastTime = timestamp;
  update(delta / 1000); // pass seconds
  render();
  requestAnimationFrame(gameLoop);
}
requestAnimationFrame(gameLoop);

Delta time makes animations frame-rate independent. Speed × deltaSeconds = consistent movement.

41. Path2D for reusable paths

const star = new Path2D('M 100 10 L 40 198 L 190 78 L 10 78 L 160 198 Z');
ctx.fillStyle = 'gold';
ctx.fill(star);
// Reuse at different positions with transform
ctx.save();
ctx.translate(200, 0);
ctx.fill(star);
ctx.restore();

Path2D stores a reusable path definition. Accepts SVG path d string directly.

42. ImageData manipulation — invert

const d = ctx.getImageData(0, 0, w, h);
for (let i = 0; i < d.data.length; i += 4) {
  d.data[i]   = 255 - d.data[i];   // R
  d.data[i+1] = 255 - d.data[i+1]; // G
  d.data[i+2] = 255 - d.data[i+2]; // B
  // d.data[i+3] = alpha — unchanged
}
ctx.putImageData(d, 0, 0);

Invert filter: subtract each channel from 255. Alpha channel is left unchanged.

43. Canvas recording with MediaRecorder

const stream = canvas.captureStream(30); // 30 fps
const recorder = new MediaRecorder(stream);
const chunks = [];
recorder.ondataavailable = e => chunks.push(e.data);
recorder.onstop = () => {

recorder.start();
setTimeout(() => recorder.stop(), 5000);

captureStream() converts canvas animation to a video stream that MediaRecorder can record.

45. Freehand drawing

let drawing = false;
canvas.addEventListener('mousedown', e => { drawing = true; ctx.beginPath(); });
canvas.addEventListener('mousemove', e => {
  if (!drawing) return;
  const r = canvas.getBoundingClientRect();
  ctx.lineTo(e.clientX - r.left, e.clientY - r.top);
  ctx.stroke();
});
canvas.addEventListener('mouseup', () => { drawing = false; });

Basic freehand drawing: track mouse down, draw lines on move, stop on mouse up.

48. WebGL triangle

const gl = canvas.getContext('webgl');
const vShader = gl.createShader(gl.VERTEX_SHADER);
gl.shaderSource(vShader, `
  attribute vec2 pos;
  void main() { gl_Position = vec4(pos, 0, 1); }
`);
gl.compileShader(vShader);
// ... link program, bind buffer, draw

Minimal WebGL pipeline: vertex shader → fragment shader → program → buffer → draw.

49. canvas.getContext options

const ctx = canvas.getContext('2d', {
  alpha: false,          // no transparency = faster compositing
  desynchronized: true,  // low-latency mode for drawing apps
  willReadFrequently: true // optimize for frequent getImageData
});

Context creation options hint at intended usage for browser optimizations.

50. Canvas accessibility wrapper

<canvas id="chart" width="400" height="300"
        role="img"
        aria-label="Bar chart: Jan 50, Feb 75, Mar 60">
  <table>
    <caption>Monthly data</caption>
    <tr><th>Jan</th><td>50</td></tr>
    <tr><th>Feb</th><td>75</td></tr>
    <tr><th>Mar</th><td>60</td></tr>
  </table>
</canvas>

Canvas is not accessible to screen readers. Add role="img" + aria-label, plus a table fallback inside the element.
