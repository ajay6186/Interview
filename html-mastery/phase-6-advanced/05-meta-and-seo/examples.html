Meta and SEO — 50 Examples

1. Character encoding

<meta charset="UTF-8">
Always the first meta tag in <head>. UTF-8 supports all Unicode characters.

2. Viewport for responsive design

<meta name="viewport" content="width=device-width, initial-scale=1.0">

3. Meta description

<meta name="description"
      content="Learn HTML from scratch with 50 hands-on exercises per topic.">
Aim for 150–160 characters. Shown in search result snippets.

4. Meta robots — allow indexing

<meta name="robots" content="index, follow">

5. Meta robots — block indexing

<meta name="robots" content="noindex, nofollow">
Use on staging environments, admin pages, or duplicate content pages.

6. Meta robots — combined directives

<meta name="robots" content="noindex, follow">
<!-- Don't index this page but follow its links -->

7. Canonical URL — prevent duplicate content

<link rel="canonical" href="https://example.com/the-definitive-url">

8. Canonical for paginated content

<!-- Page 2 of /articles -->
<link rel="canonical" href="https://example.com/articles?page=2">
<link rel="prev" href="https://example.com/articles?page=1">
<link rel="next" href="https://example.com/articles?page=3">

9. Title tag — most important SEO element

<title>HTML Mastery Guide — Learn HTML5 Fast | MySite</title>
Keep under 60 characters. Unique per page. Shown in browser tab and search results.

47. Dynamically set meta tags in SPA

function setMeta(name, content) {
  let el = document.querySelector(`meta[name="${name}"]`);
  if (!el) {
    el = document.createElement('meta');
    el.name = name;
    document.head.appendChild(el);
  }
  el.content = content;
}
// On route change:
setMeta('description', pageData.description);
document.title = pageData.title;

50. Core Web Vitals and SEO

// Google ranking factors (since 2021):
// LCP (Largest Contentful Paint) — should be < 2.5s
//   → preload hero image, serve fast server responses
// FID/INP (Interaction to Next Paint) — should be < 200ms
//   → reduce JS blocking time, defer non-critical scripts
// CLS (Cumulative Layout Shift) — should be < 0.1
//   → reserve space for images/ads, avoid inserting content above fold

// Measure with: Lighthouse, PageSpeed Insights, Chrome DevTools
