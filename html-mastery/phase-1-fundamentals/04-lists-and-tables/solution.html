HTML Glossary

DOCTYPE
Declares the document type and instructs the browser to use standards mode.
Attribute
Additional information attached to an HTML element, written inside the opening tag.
Semantic HTML
Using HTML elements that convey meaning about the content they contain, not just appearance.
Void element
An element with no children or closing tag, such as <br>, <img>, or <input>.
