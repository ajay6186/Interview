19. Table with headers attribute for complex headers

<table>
  <thead>
    <tr>
      <th id="h-name" scope="col">Name</th>
      <th id="h-q1" scope="col">Q1</th>
      <th id="h-q2" scope="col">Q2</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th id="row-alice" scope="row">Alice</th>
      <td headers="row-alice h-q1">$1.2M</td>
      <td headers="row-alice h-q2">$1.5M</td>
    </tr>
  </tbody>
</table>

The headers attribute explicitly associates a cell with its headers by ID — for complex tables with multi-level headers.

40. dl as a FAQ list

What is HTML?
HTML (HyperText Markup Language) is the standard language for creating web pages. It describes the structure of content on the web.
Is HTML a programming language?
No, HTML is a markup language, not a programming language. It does not have conditionals, loops, or functions.

<dl> is semantically appropriate for FAQ content — questions are terms, answers are descriptions.

45. Print-friendly table

<style>
  @media print {
    table { page-break-inside: avoid; }
    thead { display: table-header-group; } /* repeat header on each page */
    tfoot { display: table-footer-group; }
  }
</style>

display: table-header-group repeats the table header on each printed page. page-break-inside: avoid keeps small tables together.

46. Accessible table for screen readers — summary pattern

<!-- Legacy: summary attribute (deprecated but still read by some screen readers) -->
<table summary="Rows represent employees; columns show department and salary.">
  <caption>Employee Data</caption>
  ...
</table>

<!-- Modern: use aria-describedby with an external paragraph -->
<p id="tbl-info">Table shows each employee's department and salary.</p>
<table aria-describedby="tbl-info">...</table>

The summary attribute is obsolete in HTML5. Use <caption> or aria-describedby instead.

47. Role="list" on styled list

<!-- Some browsers (e.g. Safari) remove list semantics when list-style:none is set -->
<ul role="list" style="list-style: none; padding: 0;">
  <li>Item A</li>
  <li>Item B</li>
</ul>

Adding role="list" restores list semantics in Safari VoiceOver when CSS list-style has been removed.

48. Comparison table with aria-label on cells

<table>
  <caption>Plan Comparison</caption>
  <thead>
    <tr>
      <th scope="col">Feature</th>
      <th scope="col">Free</th>
      <th scope="col">Pro</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th scope="row">API Access</th>
      <td aria-label="API Access: not included">✗</td>
      <td aria-label="API Access: included">✓</td>
    </tr>
  </tbody>
</table>

aria-label on cells gives context to symbolic values (✓/✗) that screen readers would otherwise read as "check mark" without context.
